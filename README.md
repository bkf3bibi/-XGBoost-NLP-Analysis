專案核心介紹：法說會語意動能預測模型
1. 核心動機：挖掘文字裡的領先指標
「我觀察到投資市場多依賴財報等落後指標，但我認為企業在法說會中釋出的展望語意，是預測未來的領先指標。因此，我開發了一套 Python 自動化工具，將質化的法說會 PDF 轉化為量化的成長動能特徵，藉此預測企業未來的營收表現。」
2. 實作流程與技術挑戰 (強調解決問題的能力)
「在實作過程中，我經歷了兩次核心的技術疊代，這也讓我對數據清洗有了更深的理解：」
數據提取與清洗：
遇到的挑戰：初期提取出的詞頻非常雜亂，包含大量無意義的術語，導致模型特徵模糊。
解決方案：我重新優化了詞庫篩選邏輯，聚焦於具備產業驅動力的關鍵字（如：AI、CoWoS、伺服器），確保輸入數據具備業務含金量。
模型訓練與樣本優化：
遇到的挑戰：最初僅使用單一公司的數據，導致模型因為缺乏「對照組」而產生過度擬合（Overfitting），預測權重出現極端偏誤。
解決方案：我體會到機器學習需要數據變異性，因此擴充樣本至台積電、台達電、光寶科等多家企業，並導入 XGBoost 回歸模型。在整合 32 筆具備不同成長位階的樣本後，成功讓模型學會辨識特徵與營收成長（Target_Growth）之間的關聯。
3. 三大數據發現與價值
「透過這套流程，我得到了三個具備實戰價值的洞察：」
發現一 (量化強度)：指標 [出現次數] 的重要性佔 58.8%，證實管理層對特定領域的強調頻率與業績成正相關。
發現二 (趨勢動能)：指標 [動能變化率] 佔 41.2%，證明「今年比去年進步多少」是捕捉業績轉折點的關鍵。
發現三 (預測能力)：儘管樣本規模尚小，但模型已能精確區分 19% 至 46% 不同位階的成長率，驗證了這套語意量化邏輯具備極高的開發潛力。
